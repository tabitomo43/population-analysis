# population-analysis
PythonとSQLを使った人工データ分析

## 概要

日本では人口減少が大きな社会課題となっていますが、
より重要なのは **労働可能人口（15〜64歳）の減少** です

本プロジェクトでは以下を分析します。

- 日本の総人口推移
- 労働可能人口の推移
- 高齢化率
- GDPとの関係
- 労働生産性の推移

データ分析の流れとして

API取得 → DB保存 → SQL加工 → Python可視化

という **データパイプライン** を構築しています。
